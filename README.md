## 绪论

### 弹性力学的内容

弹性力学研究的是弹性体，学过的材料力学、结构力学还有现在的弹性力学，他们属于固体力学。

通俗来讲，材料力学研究的就是一根杆(可能是柱子，可能是梁)，结构力学研究的是几根杆(横架、钢架)，弹性力学研究对象相比而言较为广泛。

弹性力学需要从静力学、几何学、物理学三个方面设方程，在约束条件下求解方程。

材料力学是一门近似的学科,弹性力学更加严格。弹性力学是固体力学的基础，它下面还有岩石力学、弹塑力学等。

我的目的：数学上可以学会变分法、差分法、有限元法。物理上理解弹性更多。

### 弹性力学中的基本概念

外力：体力(作用于物体体积的力，比如说自重和惯性力) + 面力(作用于物体表面的力)

量纲分为基本量纲(质量M + 长度L + 时间T)和导出量纲。

内力：假想切开物体、截面两边互相作用的力。

应力：截面上某一点处、单位截面面积上的内力值。

因为力是矢量，所以力有方向，比如说应力：应力成对出现，坐标面上的应力以正面为正向，负面负向为正。

老师在这块儿画的应力的分析图，让我想到了图形学中的法线空间。

应力有正应力和切应力。

形变：形状的改变。以通过一点的沿坐标正向微分线段的正应变 $$\epsilon$$ 和切应变 $$\gamma$$ 来表示。

正应变 $$\epsilon_x$$, $$\epsilon_y$$以伸长为正；切应变$$\gamma_x_y$$ 以直角减小为正，用弧度表示。

位移用u,v,w来表示。

### 弹性力学中的基本假定

1、从静力学角度，由微分体的平衡条件，建立平衡微分方程。

2、从几何学角度，由微分线段上形变与位移的几何关系，建立几何方程。

3、从物理学角度，由应力和形变之间的物理关系，建立物理方程。

4、在给定面力的边界 $$S_{\sigma}$$ 上，建立应力边界条件。

这个好像高中数学里面爱出的一道题，给定直角坐标系内的三角形方程，然后移动直线，求什么最大面积。

1、假定研究对象是连续的，也就是没有空隙的，是被填满的。一个函数可导必须连续，连续不一定可导(比如说尖顶点，f = |x|)。

2、假定物体是完全弹性的，外力取消，变形恢复，无残余变形。

3、假定物体是线性弹性的，即应力和应变成正比，也就是应力和应变可以用胡克定律表示。

4、假定研究对象是均匀的，也就是由同种材料组成。弹性常数和位置坐标没关系，**泊松比**一样。

5、假定研究对象**各向同性**，任意点任意方向体现出来的力学性能是一样的。也就是弹性常数和方向没关系。

6、假定小变形，也就是位移和形变为很小，这样可以简化平衡方程和几何方程。
